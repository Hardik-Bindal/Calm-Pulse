# 🌊 Calm Pulse — Wellness Platform

A modern, AI-powered stress detection and wellness management platform built with React. Designed for **students** and **working professionals** to track, understand, and manage their stress patterns.

## ✨ Features

### 🎯 Core Functionality
- **Dual Mode** — Academic (Students) & Workplace (Employees)
- **Real-time Stress Score** — Gauge-based live stress monitoring
- **AI Chat** — Private, encrypted mental health chatbot
- **Stress DNA** — Unique stress pattern profiling per user
- **Smart Alerts** — Burnout warnings, overtime flags, and nudges

### 🎓 Academic Mode
- 📊 Student Dashboard with KPI cards
- 📅 AI-generated Study Planner with task tracking
- 😊 Daily Mood Tracker with weekly trends
- 🌙 Sleep Quality Logger with stress correlation
- 🧬 Stress DNA Profile
- 💬 AI Mental Health Chat (English / Hindi)

### 💼 Workplace Mode
- 📊 Work Dashboard with meeting & focus analytics
- 📅 Meeting Overload Detector
- ⚖️ Work-Life Balance Tracker
- 🗓️ Weekly Prep Briefing with AI recommendations
- 🧬 Stress DNA Profile
- 💬 AI Mental Health Chat (English / Hindi)

### 🎨 UI / UX
- 🌗 **Dark / Light Theme Toggle** — seamless switching
- 🪟 **Glass Morphism** — frosted glass cards in light mode with `backdrop-filter: blur()`
- ✨ **Smooth Animations** — fade-in, slide-in, scale-in, and pulse effects
- 📱 **Responsive Charts** — custom SVG Donut, Radar, Line/Area, Gauge, and Bar charts
- 🎯 **Consistent Theming** — all components use centralized `ThemeContext`

## 🚀 Quick Start

### Prerequisites
- **Node.js** ≥ 16.x
- **npm** ≥ 8.x

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/calm-pulse.git
cd calm-pulse

# Install dependencies
npm install

# Start development server
npm start
